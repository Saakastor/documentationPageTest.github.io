AUTH WIKI HTTP BEARER

HTTP Bearer authentication strategy for AuthWiki.

This module lets you authenticate HTTP requests using bearer tokens, as specified by RFC 6750, in your Node.js applications. Bearer tokens are typically used protect API endpoints, and are often issued using OAuth 2.0.

By plugging into AuthWiki, bearer token support can be easily and unobtrusively integrated into any application or framework that supports Connect-style middleware, including Express.
